Detailed Description

Subclass from crypt for symmetric encryption and decryption with Blowfish.

It reimplements the crypt und decrypt method from Crypt. With this Class it is easy to encrypt and decrypt data. You must only set the key or call Crypt::genRandPassword() to generate a random key. After that you can use the encrypt and decrypt methods.

Blowfish has a variable key length of up to 448 bits.
