Detailed Description

This is the settings dialog to change algorithm and see own public key.

Show the public key in a readonly PlanTaxtArea. In the Settings dialog it's possible to change the symmetric algorithmus with a Combobox. The combobox send a signal to ControllServer to change the algorihmus.
