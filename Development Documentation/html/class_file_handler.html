Detailed Description

Handle source and destination file to read an write to filesystem.

After set the input or output filepath so it check is this file exist and readable if so open the filehandle. Parse the input file path to genarate path to key file to read. After write data so it parse the output filepath to ganarate path for keyfile to write.
