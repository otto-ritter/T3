Detailed Description

Subclass from crypt for symmetric encryption and decryption with 3DES.

It reimplement the crypt und decrypt method from Crypt. With this Class it is easy to encrypt and decrypt data. You must only set the key or call Crypt::genRandPassword() to generate a random key. After that you can use the encrypt and decrypt methods.

This 3DES implementation uses the three-key method, so it needs a key with 3*56 = 168 bit key length.
