Detailed Description

Is a Exception class was inherits from MainException and is use to show User errors.

This Class has a technical and a user friendly error message to tell him the warnings. Implemented what() it will called by the standard event handler for error stacktrace. It give the user friendly errors back. If you want become the user friendly error then use the method getMessage() else getDebugMessages().
