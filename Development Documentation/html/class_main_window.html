Detailed Description

This Class represents the view for T3.

MainWindow inherits from QMainWindow and is the main view containing the friends list. Methods that were named on_xxxxxxxx_pressed or on_xxxxxxxx_triggered are autoconnected slots.

void MainWindow::on_toolButtonSelectFile_pressed ( ) [protected, slot]

Called when the user selects a file for encryption or decryption.

Shows a QFileDialog to the user to retrieve the file to process. After the file was selected check if the file suffix is ended by .enc or .key then activate decryprion button else the encryption button.

The selected filepath will be transmitted to the ControllServer directly.
