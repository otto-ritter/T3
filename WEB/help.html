Schlüssel weitergeben

Öffentlichen Schlüssel über
das Menü exportieren.

Nach dem Start des Programmes auf den Schlüsselknopf ( ) drücken. Alternativ kann dieses auch über das Menü geschehen, wie das Bild es zeigt. Durch das ausführen einer der Aktionen wird Ihr öffentlicher Schlüssel in die Zwischenablage kopiert. Zudem wird eien Datei mit der Bezeichnung "myPublic.key" im Arbeitsverzeichnis erzeugt. Den nun exportierten, öffentlichen Schlüssel kann man seinen Freunden und Bekannten per E-Mail, Instant Messenger oder im Chat weitergeben. Damit hat das Gegenüber nun die Möglichkeit, Dateien für jemanden zu verschlüsseln, die nur dieser wieder entschlüsseln kann.

Empfänger hinzufügen

Dialog um ein neuen
Fruend hinzuzufügen
Um einen neuen Empfänger hinzufügen zu können, wird sein öffentlicher Schlüssel benötigt. Sollten sie diesen noch nicht erhalten haben, setzten Sie sich bitte mit dem Empfänger in Verbindung. Um den Dialog zum hinzufügen anzuzeigen, kann entweder auf den Freund hinzufügen-Knopf ( ) geklickt werden, oder über das Menü ( Friends -> Add ). In dem Dialog kann nun in das Namensfeld eine Bezeichnung eingetragen werden, diese sollte einzigartig sein. In das Schlüsselfeld kommt nun der öffentliche Schlüssel des Empfängers. Mit einem Klick auf die Schaltfläche Save werden die Eingaben gespeichert.
Datei auswählen.

Empfänger bearbeiten

Das Bearbeiten des Namens ist nicht möglich. Nur der öffentliche Schlüssel kann geändert werden. Zuerst muss in der Liste der Empfänger derjenige ausgewählt werden, der bearbeitet werden soll. Mit der betätigen der Schaltfläche ( ) wird ein Dialog angezeigt, in dem es möglch ist, den Schlüssel zu ändern. Die andere Möglichkeit diesen Dialog anzuzeigen, ist über das Menü ( Friends -> Edit ). Sobald der Dialog offen ist, kann hier der Schlüssel des Empfängers geändert werden.

Empfänger löschen

Um einen Empfänger zu löschen, muss dieser zuerst in der Liste ausgewählt werden. Durch das drücken auf den Knopf ( ) erscheint ein Dialog, der zuerst bestätigt werden muss, um einen Empfänger zu löschen.

3. Schritt: Mit dem Klick auf die Schaltfläche ( ) wird die Verschlüsselung gestartet. Alternativ ist es möglich, über das Menü ( File -> Encrypt ) das Verschlüsseln zu starten. Sobald die Verschlüsselung abgeschlossen ist, also der Fortschrittsbalken 100% erreicht hat, befinden sich zwei weitere Dateien im selben Verzeichnis, in dem die zu verschlüsselnde Datei gewählt wurde. Die beiden Dateien Tester_Xmas2012.enc und Tester_Xmas2012.key können nun an den Empfänger geschickt werden.

Sicherung

Das Schlüsselpaar und die Freundesliste können gesichert werden. Hierzu ist es nur notwendig, den Ordner Data der von T³ erzeugt wird abzusichern.
